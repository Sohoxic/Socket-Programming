# Socket-Programming
This repository contains socket programming in C
## How to run Client-Server?
- To run `server.c`
```
gcc server.c -o server
```
```
./server 3000
```


- To run `client.c`
```
gcc client.c -o client
```

```
./client 127.0.0.1 3000
```
## To run any other .c  file
```
gcc program.c -o prg
```
```
./prg
```
